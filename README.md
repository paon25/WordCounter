# WordCounter 📝

## Description
Programme Java qui compte le nombre de mots 
dans un fichier texte ligne par ligne.

## Fonctionnalités
- Lecture d'un fichier texte avec File Scanner
- Comptage des mots par ligne
- Affichage du total de mots du fichier
- Gestion des erreurs si le fichier est introuvable

## Comment exécuter
1. Cloner le projet : `git clone https://github.com/paon25/WordCounter.git`
2. Ouvrir dans IntelliJ IDEA
3. Ajouter un fichier texte `test.txt` à la racine du projet
4. Exécuter la classe Main.java

## Résultats des tests
- Fichier test.txt : 38 mots sur 5 lignes
